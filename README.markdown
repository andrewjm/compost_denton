# Compost App

## Purpose

This app is designed to help community composting organizations with data collection.

1. Add members
2. Log each pickup weight
3. Track your waste diversion!

## Dependencies

This app is built on the following technologies:

* Ruby 2.0
* Rails 4.2.0
* PostgreSQL 9.4.2
